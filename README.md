## Description
This is a simple Java console application created to demonstrate basic version control 
workflows, including branching, merging, and remote repository synchronization.

## Project Structure
* `src/`: Contains the main Java source files.
* `README.md`: Project documentation and exercise logs.

## Commands Practiced
During this exercise, the following Git commands were used:
* `git init` - Initialize local repository
* `git checkout -b [branch]` - Create and switch to a new feature branch
* `git commit -m "[message]"` - Record changes to the history
* `git merge [branch]` - Combine work from a feature branch into main
